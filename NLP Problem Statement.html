  • Due Mar 27 by 19:00
  • Points 15
  • Submitting a file upload
  • Available Feb 27 at 19:00 - Mar 27 at 19:00

General Instructions:

  1. Follow the instructions in each question carefully.
  1. Each group is expected to submit jupyter notebook (.ipynb) with output for each cell.
  2. Submissions using other python IDEs will not be considered for grading.
  3. In case the link to dataset is not useful, same dataset can be downloaded from any online resource.
  4. A clear explanation for each output obtained is mandatory.
  5. Contribution of each member of the group must be mentioned.

Link to the Dataset: 

https://drive.google.com/file/d/1RpNUwmlio_QdgbQkhabdeoRII4-uyD7o/view?usp=sharing Links to an external site.

Description of Data:

The purpose of this analysis is to build a prediction model to predict whether a review on the restaurant is positive or negative. To do so, we will work on Restaurant Review dataset. The dataset contains two columns, "Liked" and "Review". The liked can be 0 or 1.

  1. Download the dataset and Create a dataframe named as Rest_rev, then check the head, info, and describe methods on created dataframe Rest_rev. (2 Marks)
  2. Perform pre-processing steps like Removing Punctuations, Numbers, and Special Characters, Stop Words in dataset.  (2 Marks)
  3. Normalize review by using Stemming or Lemmatization. (2 Marks)
  4. Preprocessed Review should be included in the Rest_revdata frame as cleaned_Review. Plot word cloud for all classes of the Review. (1 Mark)
  5. Create two objects Xand Y. X will be the ‘cleaned_Review ' column of Rest_rev data frame and Y will be the ' liked' column. ( 5 Marks)
    1. Create a TF-IDF object and split the data into training and testing sets. Train a MultinomialNB model. Display the confusion Matrix for both train and test.
    2. Create a Word2Vec object and split the data into training and testing sets. Train a MultinomialNB model. Display the confusion Matrix for both train and test.
    3. Compare TF-IDF and Word2Vec. Answer without justification will not be awarded marks.
  6. Display Build and display a dependency parser tree for 3 sentences. ( 3 Marks)
